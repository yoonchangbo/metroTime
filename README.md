# 지하철 탐색

## metroTime
metroTime은 지하철 최단거리 시간 및 경로 탐색 서비스입니다.  

<br>

## 사용한 알고리즘 및 자료구조
1. 알고리즘 : 다익스트라 알고리즘(다이나믹 프로그래밍 응용)
2. 자료구조 : 연결리스트

<br>

## 기능

1. 지하철역의 정보를 보여줍니다.
2. 지하철역을 2개 입력하면 최단경로를 보여줍니다.
3. 다익스트라 알고리즘에서 사용되는 최단경로표를 보여줍니다.
